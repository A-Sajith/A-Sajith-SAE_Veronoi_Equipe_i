# SAE Diagramme de Voronoï — Équipe i

##  Lancer le programme

Pour exécuter le code de la phase 1, deux méthodes sont possibles.

### Méthode 1 — Se placer dans le dossier `phase1`

1. Ouvrir un terminal
2. Se déplacer dans le dossier `phase1`.

```bash
cd A-Sajith-SAE_Veronoi_Equipe_i/phase1
python voronoi.py
```

### Méthode 2 — Lancer directement le fichier depuis la racine

```bash
python A-Sajith-SAE_Veronoi_Equipe_i/phase1/voronoi.py
```

---
Prérequis

* Python 3.x installé
* Les bibliothèques nécessaires installées (si besoin : pip install -r requirements.txt)
